# Singleton Config Preview

Java Swing 기반의 설정 관리자 미니 프로젝트입니다.  
SSAFY 스터디 목적으로 싱글톤(Singleton) 패턴을 **전역 상태 공유 관점에서 체감**하기 위해 제작했습니다.

## 🔍 What this project is
- 앱 전역에서 공유되는 설정(Config)을 하나의 객체로 관리
- 설정 변경 시 여러 화면이 동시에 반응하는 구조
- Singleton 패턴의 필요성과 효과를 직관적으로 확인

## 🧩 Key Concepts
- Singleton Pattern
- Global State Management
- Java Swing UI
- Separation of Concerns

## 🛠 Tech Stack
- Java
- Java Swing
- Eclipse IDE

## 🎯 Goal
- `new` 없이 하나의 객체를 공유하는 구조 이해
- 싱글톤을 **왜 써야 하는지**를 코드와 UI로 확인

## 🚀 How to Run
1. Clone this repository
2. Open the project in Eclipse
3. Run `MainFrame.java`
